# commands

All commands require you to install strongloop as a global

```
npm i -g strongloop
```

## arc

Launches the StrongLoop GUI, let's you define loopback models and more. Also includes profiling tools.

```
slc arc
```

## loopback:app

Creates an app.

```
slc loopback:app [<name>]
```
